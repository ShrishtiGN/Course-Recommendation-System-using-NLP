# Course Recommendation System using NLP

## Project Description
This project leverages Natural Language Processing (NLP) techniques to recommend courses to users based on their interests and preferences. The system analyzes user input and suggests relevant courses from a variety of domains.

## Features
- Intelligent course recommendations based on user input.
- User-friendly interface to interact seamlessly.
- Comprehensive data analysis to enhance accuracy.
- Supports multiple domains including technology, business, arts, and more.

## Installation
### Prerequisites
- Python 3.x
- pip

### Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/ShrishtiGN/Course-Recommendation-System-using-NLP.git
   cd Course-Recommendation-System-using-NLP
   ```
2. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
1. Run the main application:
   ```bash
   python main.py
   ```
2. Follow the on-screen instructions to receive course recommendations.
